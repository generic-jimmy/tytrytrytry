Common issues & fixes

Bot doesn't respond at all
Check telegram_webhook.url above. It must match configured_url and be reachable from the public internet. If last_error_message is set, Telegram has been failing to deliver — read the message.
Bot responds to /start but not group messages
Open @BotFather → /setprivacy → select your bot → Disable. Until privacy mode is off, the bot can only see commands (messages starting with /), not regular group messages.
Bot was added to group but does nothing
The bot must be promoted to admin in the group for moderation commands to work. Also, run any admin command (like /bwarn as a reply) once — that's what registers you for dashboard access.
pending_update_count is high
If this number is growing, Telegram is trying to deliver updates but your webhook is returning non-2xx or timing out. Check server logs for Webhook handler error entries.
Dashboard shows "Page not found"
This was a script-load-order bug in earlier versions — fixed. If you still see it, hard-refresh the browser (Ctrl+Shift+R / Cmd+Shift+R) to bypass cache.
